## 🛠️ Using the features

The tool includes modules for different tasks. You type simple commands into the window to start a scan. To see a list of available actions, type help and press Enter.

### Finding Subdomains
The software finds hidden addresses on your website. This helps you map your digital presence. Type `scan subdomains -target [your-website]` to begin. The engine uses smart guessing to find addresses other tools miss.

### Scanning for Weaknesses
The tool checks your systems for known flaws. It identifies outdated software and configuration errors. Type `scan vulnerabilities -target [your-website]` to run a check. The tool produces a clear report once the scan finishes.

### Monitoring Infrastructure
You can use the tool to watch your cloud systems. It tracks changes in your network setup. This ensures no unauthorized changes occur on your platforms.

## 📝 Preparing your environment

Some scans work best when you provide authorized lists. You can create a simple text file with the names of the websites you own. Place this file in the same folder as the program. You then point the program to this file during the scan command. This saves time and keeps your testing organized.

## 📊 Viewing your reports

The tool saves all results in a folder named Output within the main directory. You find files formatted as text or readable reports. These files list the findings from each scan. You can open them with any standard document reader. The program updates these files every time you run a new scan.

## 🛡️ Best practices for testing

Only test systems you own or have permission to scan. Unauthorized testing is illegal and causes issues for network owners. Always check your site settings before you start an automated scan. If you scan a live environment, start with a small test to ensure the system handles the traffic without slowdowns.

## 🔍 Troubleshooting common issues

If the window closes immediately, ensure you have the correct Windows version. Verify that your antivirus program does not block the software. If a scan fails, check your internet connection and verify that the target address is correct. You may need to restart the program if a scan hangs for a long time. For deep issues, make sure your computer has the latest Windows updates installed.
